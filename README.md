## Hellhouse Minion

A moderation bot for the [Hellhouse Discord](https://hellhouse.xyz)

### Setup

Setup the config file in `src/` with the name `config.json`

```json
{
  "bot": {
    "token": "",
    "defaultPrefix": ""
  },
  "database": {
    "mongoIp": "",
    "mongoUser": "",
    "mongoPass": "",
    "mongoDbName": ""
  }
}
```
